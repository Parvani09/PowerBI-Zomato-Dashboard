# PowerBI-Zomato-Dashboard
This project analyzes Bangalore restaurant data from Zomato using Power BI. The dashboard provides insights into restaurant ratings, online ordering trends, booking availability, cuisines, and location-based analysis.
# Zomato Bangalore Restaurant Data Analysis

## Tools Used
* Power BI
* Excel
* Data Visualization

## Dashboard Features
* KPI Cards
* Donut Charts
* Treemap
* Interactive Filters
  
## Key Insights
* Most restaurants support online ordering.
* BTM has the highest number of restaurants.
* Higher-rated restaurants generally have higher average costs.
